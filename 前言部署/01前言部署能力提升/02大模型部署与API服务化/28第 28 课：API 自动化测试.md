## 本课学什么

本课把手动测试变成可以重复执行的测试代码。

通过一次命令，自动验证：

- 健康检查。
- 缺少 API Key。
- 请求格式错误。
- 正常聊天请求。

## 安装测试工具

~~~powershell
python -m pip install pytest httpx
~~~

- pytest：运行测试。
- httpx：为 FastAPI TestClient 提供 HTTP 测试能力。

## 创建测试文件

~~~powershell
notepad test_main.py
~~~

测试代码：

~~~python
import os
from unittest.mock import Mock, patch

os.environ["OLLAMA_URL"] = "http://localhost:11434/api/generate"
os.environ["MODEL"] = "qwen2.5:3b"
os.environ["API_KEY"] = "test-key"

from fastapi.testclient import TestClient
from main import app


client = TestClient(app)


def test_health():
    response = client.get("/health")

    assert response.status_code == 200
    assert response.json()["status"] == "ok"


def test_missing_api_key():
    response = client.post(
        "/chat",
        json={"message": "你好"}
    )

    assert response.status_code == 401


def test_invalid_request():
    response = client.post(
        "/chat",
        headers={"x-api-key": "test-key"},
        json={}
    )

    assert response.status_code == 422


@patch("main.requests.post")
def test_chat_success(mock_post):
    mock_response = Mock()
    mock_response.raise_for_status.return_value = None
    mock_response.json.return_value = {
        "response": "测试回答",
        "model": "qwen2.5:3b",
        "done": True
    }

    mock_post.return_value = mock_response

    response = client.post(
        "/chat",
        headers={"x-api-key": "test-key"},
        json={"message": "你好"}
    )

    assert response.status_code == 200
    assert response.json()["reply"] == "测试回答"
    assert response.json()["model"] == "qwen2.5:3b"
    assert response.json()["done"] is True
    mock_post.assert_called_once()
~~~

## 运行测试

~~~powershell
pytest -q
~~~

实际结果：

~~~text
4 passed, 3 warnings
~~~

4 passed 表示四个测试全部通过。warnings 是 FastAPI 提示 on_event 写法未来需要升级，不影响当前功能。

## Mock 的作用

正常聊天测试没有真正调用 Ollama，而是用 Mock 模拟 Ollama 的返回结果。

这样做的好处：

- 测试速度快。
- 不占用模型资源。
- 不依赖 Ollama 当时是否运行。
- 可以专门测试 FastAPI 自己的处理逻辑。

自动化测试让项目可以重复验证，不需要每次手动打开 Swagger。
