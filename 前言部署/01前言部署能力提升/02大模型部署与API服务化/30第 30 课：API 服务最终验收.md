## 最终验收项目

### 1. 自动化测试

~~~powershell
pytest -q
~~~

预期：

~~~text
4 passed
~~~

### 2. Ollama 模型

~~~powershell
ollama list
~~~

确认 qwen2.5:3b 存在。

### 3. 端口

~~~powershell
Get-NetTCPConnection -LocalPort 8000 -State Listen
Get-NetTCPConnection -LocalPort 11434 -State Listen
~~~

确认 FastAPI 和 Ollama 都在监听。

### 4. 健康检查

~~~powershell
Invoke-RestMethod -Uri "http://127.0.0.1:8000/health"
~~~

预期：

~~~json
{
  "status": "ok"
}
~~~

### 5. Swagger 聊天测试

打开：

~~~text
http://127.0.0.1:8000/docs
~~~

使用正确 x-api-key 和正常问题，预期：

- 状态码 200。
- reply 有模型回答。
- model 为 qwen2.5:3b。
- done 为 true。

### 6. 异常测试

- 错误 API Key：401。
- 缺少 message：422。
- 超过请求限制：429。
- Ollama 不可用时：502 或 504。

## 安全提醒

验收截图中曾显示完整 API Key。该 Key 应视为已经暴露，不再用于正式环境。

轮换方式：

~~~powershell
Ctrl+C
$env:API_KEY = python -c "import secrets; print(secrets.token_urlsafe(32))"
powershell -ExecutionPolicy Bypass -File .\start.ps1
~~~

不要把 API Key 写进公开代码仓库，也不要在截图中显示完整 Key。
