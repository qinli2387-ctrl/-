## 阶段验收结果

- FastAPI 运行在 8000 端口。
- Ollama API 运行在 11434 端口。
- 能通过 API 调用 qwen2.5:3b。
- 正常请求返回 200。
- 错误 API Key 返回 401。
- 错误请求格式返回 422。
- 超过请求次数返回 429。
- Ollama 调用错误能够返回 502 或 504。
- pytest 自动化测试通过。
- 启动脚本可以加载项目配置并启动服务。
